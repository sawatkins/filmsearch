Welcome! FilmSearch is an AI-assisted tool to discover movies using natural language. I made it to discover movies similar to ones I like or match specific moods or interests. I hope you find it useful too!

It uses OpenAI's GPT-4o model to understand natural language queries and find relevant movies. Then it uses The Movie Database API to get movie posters.

Privacy

I don't collect personal information, store cookies, or use tracking. I log search queries improve the results (and I'm curious what people search for).

Keep in mind that search queries get sent to OpenAI's API for processing, so refer to their privacy policy.
