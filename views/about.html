Welcome! FilmSearch is an AI-assisted tool to discover movies using natural language. It's a personal project of mine. It can be useful to discover movies similar to ones you like or match specific moods or interests.

It uses OpenAI's GPT-4o model to understand natural language queries and find relevant movies. Then it uses The Movie Database API to get movie posters. The backend is written in Go.

Feedback & Contact

For anonymous feedback and suggestions, use this form. For anything else, feel free to email me.

Privacy

I don't collect personal information, store cookies, or use tracking. I log search queries (because I'm curious what people search for).

Keep in mind that search queries get sent to OpenAI's API for processing, so refer to their privacy policy.
